
# HouseChat V1.0

Group Chats coming soon !

## Instructions:
* Clone this repository
* Open a terminal window in your new downloaded directory
* Type **pod install**
* Run the project in Xcode
* You can add me as a friend and we can talk, **sam@y.com** 
* Enjoy

## Feature List:
* **Send Messages** : This app facilitates two-way, real-time communication between users
* **Sent Status** : This app lets you know if your message has been viewed(its not possible to disable this for now)
* **send Images and Videos** : This application allows you to send media content(Images/Videos) to your contact
